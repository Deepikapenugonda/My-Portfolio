### 3. HR MIS Workforce Analytics Dashboard
- Imported and analyzed real HR dataset from Kaggle 
  (17,418 employee records)
- Built Analysis sheet with 5 key HR metrics using 
  COUNTIF and AVERAGEIF formulas
- Created interactive Dashboard with 5 charts covering:
  → Department wise employee count
  → Recruitment channel mix
  → Gender distribution
  → Education level distribution
  → Average age by department
- Tools used: Google Sheets, Kaggle Dataset
